class cwe.model.ModelPackage
extends cwe.model.ModelElement


A Package for structuring Model Classes.

This class is not intended to be instantiated, but to be extended.

Only one instance of each concrete subclass must exist. They are registered to cwe.model.ModelPackageContainer and can be retrieved from there by their unique id.
