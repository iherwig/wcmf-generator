
To run the Codegenerators from this folder, start here: BuildingAndRunningCodegenerators.odt (olymposSVN/trunk/olympos/chronos/docs)

Scripts:
- run_grails.sh : run the grails cartridge 
	input: 			insert-new-movie-with-api.uml
	output folder: 	referenceWSDLs/grails-gen/grails
	
- run_wsdl.sh : run the wsdl cartridge 
	input: 			insert-new-movie-with-non-compliant-api.uml
	output folder: 	referenceWSDLs/wsdl-gen
