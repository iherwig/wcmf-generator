Welcome to the Chronos Web Modeler 0.9.9.5 Beta Release.

The CWM is a tool for:

  • Collaborative modeling of Requirements, Use Cases and Domain.
  • Direct generation of Documents & Code.
  • Integration with other UML tools .



The CWM is:

  • Open
  • Extensible
  • Free

How to start

  1. Use the provided model Chronos Default model to start
  2. Check out the example Eurocom Model
  3. Read the help

Beta version limitations

This is a Beta version, for your evaluation only. Not all features are presents and some are not stable. If you encounter any issues, please report them in our bug tracking system. Additional feature requests may be reported using our feature tracking system.
